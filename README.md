<h2><a class="anchor" id="overview"></a>Overview</h2>

This project provides a comprehensive analysis of IT companies' stock performance from 2016 to 2025.  
The study focuses on understanding price movements, volatility patterns, and company-wise growth trends.

Using Python for data cleaning and analysis, and Power BI for dashboard creation, the project transforms raw stock data into meaningful insights that help understand market behavior and long-term growth patterns.

---

<h2><a class="anchor" id="problem-statement"></a>Problem Statement</h2>

Investors often find it difficult to interpret stock market fluctuations due to volatility and complex financial metrics.

- Without proper analysis, identifying stable and growing IT companies becomes challenging.
- This project simplifies stock data into clear visual insights.
- It helps in understanding long-term trends and comparing company-wise performance effectively.

---

<h2><a class="anchor" id="tools-technologies"></a>Tools and Technologies</h2>

- **Python:** Data cleaning and Exploratory Data Analysis (EDA)
- **Pandas & NumPy:** Data manipulation and calculations
- **Matplotlib & Seaborn:** Data visualization
- **Power BI:** Interactive dashboard creation
- **Microsoft Excel:** Initial data preparation
- **GitHub:** Version control and project documentation

---

<h2><a class="anchor" id="method"></a>Methodology</h2>

The project followed a structured analytical approach:

1. **Data Collection:** Collected historical stock price data of major IT companies.
2. **Data Cleaning:** Handled missing values and structured the dataset.
3. **Exploratory Data Analysis:** Analyzed yearly trends, price distribution, and volatility.
4. **Comparative Analysis:** Compared company-wise Open, Close, High, and Low prices.
5. **Dashboard Development:** Created interactive Power BI dashboards for visualization.

---

<h2><a class="anchor" id="dashboard"></a>Dashboard</h2>

The Power BI dashboard includes:

- Year-wise Open vs Close price comparison
- Average High and Low price analysis
- Close price distribution
- Company-wise performance comparison
- Turnover and share trend visualization
